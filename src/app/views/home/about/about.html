What is this?

This is a site designed to make creating Splatfests for Splatoon 1 much easier. Its main purpose is to simplify and speed up the process of setting up custom splatfests for the Splatoon community. (Of course this site is unofficial!)

The tool was originally developed for the Splatfestival team but was later made publicly available due to its usefulness and the community’s interest. Additionally, you can find its source code here.

How was this made?

This was made using Angular and Tailwind CSS with no need of a backend. The design is heavily inspired by Splatoon3.ink and official splatoon websites.
